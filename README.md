## Getting Started

### Installation  
  1. [Download and install Conda](https://conda.io/docs/download.html)
     * Install conda enviroment:
     
     	* Ubuntu: 
            ```
            conda env create -f environment.yml
            ```
     		Then:
     		```
     		source activate bearing_env  
     		```
     	
        * Windows:
     		```
     		conda env create -f environment.yml
     		```
     	Then:
     		```
            conda activate bearing_env
     		```
